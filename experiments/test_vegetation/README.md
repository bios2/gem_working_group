README.md

In this project we develop a new model inspired by the vegetation model in Madingley. The growth and mortality dynamics follow essentially the same rules, but we also implement regional heterogeneity caused by natural disturbances such as fire and windthrow. The model describes the local and regional dynamics of herbs, evergreen trees and deciduous trees. It adds competitive interactions so that the fraction of biomass made of the different groups is an emerging property instead of a fixed one like in Madingley. 

We hope this new model will solve a series of problems we found with the original model. More specifically, our basic expections are : 

- there are three types of vegetation : open herbs, coniferous trees, deciduous trees

- biomass dynamics are affected by forest fires

- herbivory decreases amount of vegetation available for consumption

- density-dependence of vegetation

- the equilibrium composition of vegetation changes with precipitation and temperature
